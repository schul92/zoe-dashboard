üé® DSL Syntax Quick Reference

These diagrams use a simple Domain-Specific Language (DSL):

  • (Label) - Ellipse (Start/End points)
  • [Label] - Rectangle (Process steps)
  • {Label?} - Diamond (Decisions)
  • [[Label]] - Database (Data storage)
  • -> - Arrow (Normal flow)
  • -> "text" -> - Labeled Arrow (Conditions)
  • --> - Dashed Arrow (Annotations)
